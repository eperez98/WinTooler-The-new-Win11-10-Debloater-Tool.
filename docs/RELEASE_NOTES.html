Visual editor with ad-block and privacy presets. Lazy-loads on first visit. Backup/Restore.

Driver Updater (Build 5045)

Background WMI scan, outdated detection, winget update integration.

NEW

Performance Benchmarks (Build 5045)

System.Diagnostics.Process.Start() for WinSAT — inherits admin token, writes to DataStore. Two-layer XML parse with regex fallback.

NEW

Registry Cleaner (Build 5045)

Three-pass orphaned key scan. Per-item selective clean. reg.exe backup.

NEW

WSL Manager (Build 5045)

Both wsl commands in Start-Job with UTF-8 encoding fix + null-byte strip. DispatcherTimer polling. No UI-thread blocking.

NEW

Custom Tweak Builder (Build 5045)

Form-based registry tweak creator. Apply/Undo. Export/Import JSON. config\custom_tweaks.json persistence.

FIX

WSL Refresh crash (Build 5045)

Four root causes: UTF-16 null bytes, synchronous network call on UI thread, RaiseEvent wrong constructor, inline-if in scriptblock. All resolved.

FIX

WinSAT "No data found" (Build 5045)

Start-Job lost admin token. Replaced with System.Diagnostics.Process.Start() + 1.5s flush + DOM/regex parse fallback.

FIX

ISO app embedding Object[] crash (Build 5045)

StringBuilder + File.WriteAllText + SelectSingleNode + XmlNamespaceManager.

FIX

ISO app picker category filter (Build 5045)

All picker state moved to $script:-scoped variables. Category pills now filter correctly.
