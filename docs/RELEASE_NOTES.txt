OVERVIEW
--------
Build 5046 is a targeted expansion release built on top of the V0.8 Power
Features drop (Build 5045). The headline change is the integration of
Chris Titus Tech's WinUtil tweak catalogue, which grows the System Tweaks
catalog from 23 to 58 entries and adds a new "Advanced" category covering
network, display, Copilot, OneDrive, Edge, and more. Templates (Standard,
Minimal, Heavy) have been fully updated to reflect the new catalog.
All Build 5045 Power Tools ship unchanged.

CARRIED FROM BUILD 5045 — POWER TOOLS
--------------------------------------

[NEW] Hosts File Editor
  Visual editor for C:\Windows\System32\drivers\etc\hosts.
  Ad-Block Preset (10 networks) + Privacy Preset (8 MS telemetry domains).
  Backup/Restore, Add Entry form, inline delete. Lazy-loads on first visit.

[NEW] Driver Updater
  Win32_PnPSignedDriver WMI scan in Start-Job. Flags drivers > 3 years old.
  Update Selected (winget) + Update All (elevated window).

[NEW] Performance Benchmarks
  System.Diagnostics.Process.Start() launches winsat.exe directly — no
  Start-Job — so WinSAT inherits the full admin token and writes to
  DataStore. DispatcherTimer polls HasExited. Two-layer XML parse: XmlDocument
  DOM first, regex fallback.

[NEW] Registry Cleaner
  Three passes: HKLM Uninstall missing UninstallString, HKCU Run pointing
  to absent executables, HKLM Run pointing to absent executables.
  reg.exe export backup before clean. Per-item selective removal.

[NEW] WSL Manager
  Both wsl --list --verbose and wsl --list --online run inside Start-Job
  with [Console]::OutputEncoding = UTF8 and null-byte strip to handle
  WSL's UTF-16 output on Windows 11. DispatcherTimer polls completion.
  No synchronous UI-thread blocking.

[NEW] Custom Tweak Builder
  Form: Name / Description / Risk / RegPath / ValName / ValType / Apply /
  Undo. Saves to config\custom_tweaks.json. Export/Import JSON.
  Lazy-loads saved tweaks on first visit.


BUG FIXES (Build 5045 / 5046)
------------------------------

[FIX] WSL Refresh crash
  Root causes: (1) wsl.exe UTF-16 null bytes crashing string operations,
  (2) wsl --list --online synchronous network call blocking the WPF STA
  thread, (3) RaiseEvent() wrong RoutedEventArgs constructor, (4) inline-if
  in stored scriptblock. All four fixed.

[FIX] WinSAT "No WinSAT data found" after benchmark
  Start-Job spawned a new PowerShell.exe that lost the admin token context
  needed to write to DataStore. Replaced with
  System.Diagnostics.Process.Start() (non-blocking direct child process) +
  1.5s flush delay + XmlNamespaceManager DOM parse + regex fallback.

[FIX] Chocolatey auto-install in App Manager
  When choco.exe absent and app has Chocolatey ID, the install job
  now downloads community.chocolatey.org/install.ps1, runs it, updates
  PATH, and retries. No manual step needed.

[FIX] ISO app embedding Object[] to UInt32 crash
  StringBuilder + File.WriteAllText replaces array += / Out-File.
  SelectSingleNode + XmlNamespaceManager replaces Where-Object on
  XmlDocument.settings (which returned Object[]).

[FIX] App picker category filter not working
  WPF mouse event handlers run in script scope. All picker state moved to
  $script:ISOPopCurCat, ISOPopRows, ISOPopLocalSel, etc.
